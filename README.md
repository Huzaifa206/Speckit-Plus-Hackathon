## ðŸ“š Overview

The Speckit-Plus-Hackathon project is an educational platform focused on **Physical AI & Humanoid Robotics**. It serves as a comprehensive learning resource that bridges the gap between abstract AI models and their embodiment in physical robotic systems. The platform uses a simulation-first approach to safely and efficiently develop robotic systems before real-world deployment.

This project was created as part of a hackathon submission, demonstrating the capabilities of the Spec-Driven Development (SDD) framework and the "Speckit-Plus" tooling for creating structured, well-documented educational content about Physical AI and Humanoid Robotics.

## ðŸŽ¯ Core Principles

The project is built around 6 core principles:

1. **Simulation-First, Hardware-Second** - Development begins in simulation
2. **AI + Robotics Convergence** - Seamless integration of AI and robotics
3. **Human-Centered Humanoid Design** - Prioritizing human-like interaction patterns
4. **Progressive Complexity** - Structured learning from simple to complex
5. **Industry-Grade Tooling** - Using professional-grade tools (ROS 2, Isaac Sim)
6. **Learn by Building** - Every concept results in tangible, executable code

## ðŸ“– Educational Content

The curriculum is structured around foundational concepts:

### Chapter 1: Foundations
1. **From Digital AI to Physical Intelligence** - Understanding the transition from abstract AI models to embodied agents
2. **The Robotic Nervous System (ROS 2)** - Mastering the communication framework for humanoid robots
3. **Digital Twins: Simulating Reality** - Creating virtual environments for testing and development

Each lesson includes hands-on labs and practical exercises that produce visible results, following the "Learn by Building" philosophy.

## ðŸš€ Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn
- Python (for ROS 2 integration)

### Installation

1. Clone the repository:
```bash
git clone https://github.com/your-username/Speckit-Plus-Hackathon.git
cd Speckit-Plus-Hackathon
```

2. Navigate to the website directory:
```bash
cd website
```

3. Install dependencies:
```bash
npm install
```

4. Start the development server:
```bash
npm start
```

5. Open your browser to [http://localhost:3000](http://localhost:3000) to view the documentation.
